# Hand Gesture Drawing Canvas 🎨

## 📌 Description

This project is a real-time hand gesture-based drawing system that allows users to draw on a virtual canvas using their fingers. It uses computer vision techniques for hand tracking and gesture recognition.

## 🚀 Features

* Draw using index finger
* Erase using two fingers
* Real-time hand tracking
* Touchless interaction

## 🛠 Technologies Used

* Python / JavaScript
* OpenCV / MediaPipe
* TensorFlow.js

## ▶️ How to Run

1. Install dependencies (for Python version):

```
pip install opencv-python mediapipe numpy
```

2. Run:

```
python main.py
```

OR open:

```
hand_drawing_app.html
```

## 🎯 Use Cases

* Touchless drawing system
* Gesture-based UI interaction
* Computer vision learning project
